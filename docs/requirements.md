# 🏢 SaaS Gestión de Fincas - Plan de Desarrollo con Codex

## 🎯 Objetivo
Construir un MVP rápido de un SaaS para inmobiliarias/administradores de fincas que permita:
- Centralizar propiedades
- Gestionar contactos
- Gestionar incidencias
- Enviar comunicaciones masivas
- Gestionar documentos
